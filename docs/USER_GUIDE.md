# AI 影视 3D 导演台使用指南

## 1. 启动导演台

macOS 可双击 `启动3D导演台.command`。源码环境执行：

```bash
npm install
npm run dev
```

等待终端打印 `Director desk` 后再打开页面。请保持终端或桌面程序运行。

## 2. 搭建场景

- 使用底部工具栏添加角色、群演、基础几何体、模型和摄像机。
- 新基础体包括胶囊体、圆角盒、楔形、平面、圆盘、管道和半球。多选对象后可在模型属性面板执行“组合所选”。
- 在左侧对象树选择对象，右侧属性面板修改位置、旋转、缩放、姿势和动作。
- 组合内的部件使用局部坐标；“旋转轴”可把门轴、车轮轴或翅膀根部设为旋转中心。
- 对象树中的删除按钮或 `Delete / Backspace` 可删除选中对象。
- 场景设置中可打开或关闭地面网格。
- 导入 FBX、OBJ、GLB 或全景背景后，使用完整工程 JSON 保存这些资产信息。

## 3. 使用 AI 快速布景

打开 AI 布景面板，将符合 `apply_scene_script` schema 的 JSON 粘贴后执行。建议 Agent 先描述角色关系、道具关系和构图，再一次性生成命令。

执行前需要完整备份时先导出工程 JSON。布景完成后可导出布景命令，用于一键恢复可表达的角色、几何体、机位和轨迹动画。

布景命令只调用白名单场景工具，不接受任意 JavaScript。

复杂物体应使用 `groups[].children` 描述部件树。`repeat` 用于阵列，`mirror` 用于镜像，`pathCopy` 用于沿路径复制。执行后导演台会返回截图，Agent 应检查轮廓、比例、站位和潜在穿插，再提交 `update_prop` 修正。

## 4. 统一动画时间轴

- 在对象树按住 `Shift` 多选两个以上道具，点击“组合所选道具”。组合后的整体可移动、缩放、删除和添加动画。
- 点击视口底部工具栏的动画图标，展开统一时间轴。角色、组合物体和子部件在同一播放头下同步。
- 播放方式只有“手动播放”“录制时播放”“随镜头运动”三种；“循环播放”是独立开关。
- 单个物体仍可在“动画设置”中手动记录关键帧和路径；多人或多部件联动优先导入 AI 动画序列。
- “关键帧动画”：先设置记录时间，再摆放物体，点击“记录当前位置 / 旋转 / 缩放”。至少记录开始和结束两个时间点。
- “路径点动画”：选择直线或平滑曲线，移动物体后点击“把当前物体位置添加为路径点”。路径点按编号显示，可直接修改坐标或删除；中央导演视角会显示蓝色轨迹线。
- 动画循环时长固定为 5 / 10 / 15 秒，可使用位置、旋转、缩放关键帧或直线/曲线/闭合路径。
- 组合动画会带动全部子部件；子部件仍可独立旋转，例如车轮、龙翼、螺旋桨和门板。
- 带内置动画的 GLB/FBX 继续播放模型动画，外层组合仍可叠加路径移动。

## 5. 手机加入与多人机位

1. 确保手机与电脑位于同一 Wi-Fi 或可互访网络。
2. 扫描导演台二维码，统一进入手机控制页面。
3. 首台手机绑定一个机位；新手机优先绑定空闲机位，没有空闲机位时自动创建。
4. 手机顶部“我的镜头”栏可切换该手机有权控制的机位。
5. 多人同时使用时，不要在多台手机上选择同一个已被占用机位。

进入全屏后页面会请求横屏锁定。如果浏览器不允许锁定方向，导演台会自动旋转为横屏操控布局。退出全屏后恢复普通页面。

## 6. 手机摄影机控制

- 左侧摇杆：前进、后退、左移、右移。
- 直接拖动监看画面：调整镜头水平和俯仰方向。
- 高度：控制摄影机升降。
- 焦段：改变 FOV/变焦效果。
- 重置：恢复该机位的基准状态。
- 体感模式：在支持安全上下文和设备方向权限的手机上控制朝向；真实空间 6DoF 位移不作为可靠能力。

手机画面只用于机位监看与摄影机操作，不会选中或拖动导演台中的物体。

## 7. 角色姿势与动画

- “姿势”页可选择静止姿势、预设姿势或进入骨骼编辑。
- “动作”页可选择预设循环动作和 5 / 10 / 15 秒循环时长。
- “Codex 轻快舞”是 15 秒连续编舞，包含交叉步、抬腿、连续蹲跳、手臂过头和横向摆臂，可手动播放或改为随镜头运动。
- 随镜头运动模式下，录制机位移动时统一动画推进，机位停止约 200ms 后暂停，不会追赶或突然加速。
- 上传人物视频可提取动作；上传全身人物图片可提取单帧姿势。识别质量取决于全身可见程度、遮挡和拍摄角度。
- AnimoFlow 文字动作需要可用的本地或远程 AnimoFlow 服务。
- 角色“属性”页可单独导出或导入角色 JSON。文件包含体型、颜色、骨骼姿势、动作设置、视频动作帧和可用的角色模型引用，不必导入整个布景。

## 8. 录制与导出视频

1. 在手机选择 5、10 或 15 秒。
2. 点击录制后移动镜头；录制结束会保存轨迹动画和机位监看的原始画面。
3. 在电脑端摄像机动画列表确认“原始录制视频已就绪”。
4. 点击导出 MP4。封装阶段不会重新播放一次场景。

录制期间请保持导演台窗口可见，避免浏览器或系统将渲染降为后台帧率。复杂场景可适当降低模型数量与贴图尺寸，优先保证实时监看的稳定帧率。

## 9. 保存、导入与重置

- **工程 JSON**：完整备份场景、导入资产、角色姿势、动作、机位和动画。
- **布景命令 JSON**：适合 Agent 修改和跨工程快速还原基础布景。
- **角色 JSON**：在单个角色“属性”页导出或导入，适合复用编辑过的角色与动作。
- AI 布景面板的“导入并执行”支持完整布景、独立角色、角色动画包和统一动画序列 JSON。完整布景会替换当前导演台；角色动画包只把舞者、格斗者及其动画追加到当前布景。动画序列会先显示参与对象、时长和轨道摘要，确认后再应用。
- **重置导演台**：恢复为一个默认角色和一个基础机位；不会删除已经导出的文件。

导入后等待场景资产加载完成，再开始手机录制。

供其他 AI 生成布景和角色 JSON 的完整字段、约束与可复制指令见 [导演台 JSON 生成指南](AI_SCENE_SCRIPT_GUIDE.md)。

## 10. Windows 使用与排错

Windows x64 可使用 Setup 安装版或 Portable 便携版。首次启动若被系统保护拦截，请确认文件来源后在 Windows 安全界面选择继续运行。

如果窗口没有打开，查看日志：

```text
%APPDATA%\AI影视导演台\logs\director-desk.log
```

常见检查：

- 解压或安装路径具有写入权限。
- 防火墙允许导演台在专用网络中通信，否则手机无法连接。
- 没有其他导演台实例占用本地端口。
- Windows 版本为 64 位，并已安装当前系统更新。
